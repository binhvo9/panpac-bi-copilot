# Notes:
# - Dim_Date, Dim_Product, Dim_Region are conformed dimensions

# - Fact_Production connects Operations domain
# - Fact_Shipment connects Supply Chain domain
# - Fact_Finance connects Finance domain
#
# - All fact tables use surrogate integer keys:
#   date_key, site_key, product_key, customer_key, region_key
#
# - This ERD is designed for Tableau and DuckDB
